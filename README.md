# TempConv-WebApp
A functional web application that converts temperatures between Celsius and Fahrenheit. This project focuses on clean design and simple user interaction.

## Features
* Bidirectional Conversion: Supports both Celsius to Fahrenheit and Fahrenheit to Celsius.
* Responsive Design: A centered card layout that adapts to different screen sizes.
* Visual Feedback: Includes hover effects on buttons and clear result displays.
* Clean Code: Minimalist implementation using vanilla JavaScript, HTML, and CSS.

## Tech Stack
* HTML5
* CSS3
* JavaScript (ES6)

## Calculation Logic
The application utilizes standard conversion formulas:
* Celsius to Fahrenheit: $F = (C \times 1.8) + 32$
* Fahrenheit to Celsius: $C = (F - 32) / 1.8$
